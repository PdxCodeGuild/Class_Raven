css style tags

style tags set styling rules

    
    
      
  1. sets style by html tag
    2. 
      
  2.  any rule with a . before it styles anything 
     with that class written into the tag
    3. 
      
  3.  any rule following a # will only style items 
     with that id written into it's tag
    4. 
      
  4.  anything in the curly brackets is the actual style desired
    5. 
    

      

      examples: 

      this styles ONLY/ALL p tags WITH class "styleme"
      

      p.styleme {
        color: white;
        background-color: black;
      }
      

      this would style ALL p tags
      

      p {
       color: white
      }
      

      this would style ALL tags with the class "styleme"
      

      .styleme {
        color: white
      }
      

      this styles ONLY the item with the id "stylespan"
      

      #stylespan {
        color: red
      }
      

    
    you can also write all of this to a .css file,
 and link it by writing in a path to that file in a link tag
    <link href="style.css">
